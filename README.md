# My Code Sampler

A Vite + React + TypeScript project.

## Tech Stack

- Vite
- React
- TypeScript

## Getting Started

Install dependencies:

```sh
npm install
```

Run the development server:

```sh
npm run dev
```

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview build
